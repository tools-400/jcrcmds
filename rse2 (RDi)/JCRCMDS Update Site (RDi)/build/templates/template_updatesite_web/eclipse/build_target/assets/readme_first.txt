
JCRCMDS Installation
====================


Step 1: Installation of JCRCMDS on the i5
-----------------------------------------

First you need to download and install the JCRCMDS library of Craig
Rutledge from his Internet page at http://www.jcrcmds.com/.


Step 2: Installation of the JCRCMDS plugin
------------------------------------------

Option 1: Using the update site
-------------------------------

The URL of the JCRCMDS plugin for RDi8.0+ site is: 

  http://jcrcmds.sourceforge.net/eclipse/rdi8.0/

The URL of the JCRCMDS plugin for WDSCi7.0 site is: 

  http://jcrcmds.sourceforge.net/eclipse/wdsci7.0/
