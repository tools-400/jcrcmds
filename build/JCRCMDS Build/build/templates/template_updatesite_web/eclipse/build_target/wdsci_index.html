Download Plug-In and i5 Component

Follow the installation steps given in readme_first.txt to install JCRCMDS on your system. The zip file includes a local update site for the plug-in as well. The JCRCMDS library must be separately downloaded from Craig Rutledge's homepage at http://www.jcrcmds.com/.
